│ ├── context/ → Context API (ThemeContext, etc.)
│ ├── hooks/ → Hooks personalizados
│ ├── pages/ → Home, About, Projects, Contact
│ ├── routes/ → Configuração de rotas
│ ├── utils/ → Funções auxiliares
